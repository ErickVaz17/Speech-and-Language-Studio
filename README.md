# Análise de Sentimentos com Language Studio
Um coisa que me chamou bastante atenção foi pelo Language Studio separar em frases e analisar parte por parte, e não o texto por completo, assim não obtendo uma analise completa e detalhada.

Cada sentença é uma separação de uma frase onde foi analisado e extraido os sentimentos reconhecidos.

Pode ser utilizado para verificar diversos comentários e/ou críticas sobre seu empreendimento, com esse recurso será possivel saber se os comentários e/ou críticas estão sendo positivos, negativos ou neutros.
